show-me-the-coffee — 사용 예

스크립트 한 줄 + SMTC('boot', {...}) 호출이면 우하단에 후원 버튼이 뜹니다.

<script src="show-me-the-coffee.js"></script>
<script>
  SMTC('boot', {
    kakaoPayUrl: 'https://qr.kakaopay.com/Ej84nMWnw',
    name: '임태환',
    siteKey: 'fin-term',
    analytics: true,
  });
</script>

우하단 버튼을 눌러 QR 모달을 확인하세요. 색·아이콘·문구·크기 전부 옵션으로 바꿀 수 있습니다.
